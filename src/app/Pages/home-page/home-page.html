Comment trouver mon artisan ?

(En seulement 4 étapes, facile non?)

  1. Choisissez une catégorie :

    2. Utilisez le menu pour sélectionner le type d'artisanat qui vous intéresse : plomberie,menuiserie,boulanger,électricité, etc.

  2. Trouvez votre artisan :

    3. Parcourez les profils disponibles dans la catégorie choisie. Consultez leurs fiches et choissiez.

  3. Contactez-le facilement :

    4. Remplissez le formulaire de contact présent sur sa fiche. Décrivez votre besoin, votre projet ou posez vos questions.

  4. Recevez une réponse rapide :

    5. L'artisan vous répondra sous 48 heures maximum. Simple, rapide et efficace !
